Historia

En 1997 Netscape inventa JavaScript con la finalidad de controlar el navegador y añadir dinamismo e interactividad a las páginas Web.

¿Qué cosas puedo hacer con Javascript?

JavaScript permite crear una interfaz de usuario activa

Llamadas asincronicas a un servidor (llamadas ajax)

¿Qué más?

  • Validar la información que se puso en formularios
  • Manejo de Cookies
  • Redirecciones y manejo del navegador desde el cliente
  • Manejo de Session y Local Storage
  • Manejo de canvas (HTML5) Ejemplos
  • Modificaciones de DOM (Document Object Model)
  • Creación de Código de servidor con la plataforma NodeJs. Ejemplo

¿Qué cosas NO puedo hacer?

  • No permite la escritura de archivos en las máquinas servidor
  • No puede cerrar una ventana que no ha abierto

DOM (Document Object Model)

El DOM permite un acceso a la estructura de una página HTML mediante el mapeo de los elementos de esta página en un árbol de nodos.
